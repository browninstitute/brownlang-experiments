6 - An on-the-go map-reduce tester

Map-reduce are important operations in functional programming. In particulart, it can be super common when working with data tables (using lambda functions that take each row as argument.)
We could make a tester that would display in real time the 5 first rows of the previous object and the 5 first rows of the next object, so that the user would know in real-time what they are doing.
