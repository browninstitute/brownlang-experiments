4 - A function inspector

Same as example 1, but with a function. When the user writes a function and uses it later, we could unfold it, and potentially compute the values of the used variables.

Click on the icon to unfold the function.
Click on the icon to go further in the inspection by computing the values.

def greatest_element (table1, table2):
max_element = None
for element in table1:
if element > max_element:
max_element = element
max_element1 = max_element
for element in table2:
if element > max_element:
max_element = element
if max_element1 == max_element:
return table1
else
return table2


list1 = [3, 5, 7]
list2 = [1, 2, 8]
list3 = greatest_element(list1, list2)
