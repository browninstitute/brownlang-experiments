1 - A loop inspector

The idea is pretty simple here. In order to make loops more tangible, the GUI offers the option to see a loop in its entirety.

When the user clicks on the icon next to a loop instruction, the loop "unfolds". What that means is that all the instructions executed multiple times inside the loop are displayed. Also, if there an iteration variable (i in this case), it is automatically converted to the real value.

After the loop is unfolded, when the user clicks on the icon, the program computes the values of the variable changed in the loop in order to see the evolution of theses variables.

You can click on both the icon and the icon in this demo.
